SonosESP Controller

Web Firmware Installer

v0.1.0

Hardware Requirement: GUITION JC4880P433C (800x480 RGB, GT911 Touch, ESP32-P4 + ESP32-C6)

Hardware Specifications

Microcontroller ESP32-P4
Display 800×480 RGB
Touch GT911 I2C
Flash 16 MB
PSRAM OPI

Installation Steps

  1. Connect Device Plug ESP32-P4 via USB-C
  2. Flash Firmware Click button below, select COM port
  3. Wait for Completion Do not disconnect during flash
  4. Configure WiFi Use on-screen keyboard after reboot

Requires Chrome, Edge, or Opera
